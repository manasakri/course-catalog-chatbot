# Course Catalog Chatbot

A Go-based chatbot that lets users ask questions about course information, like class descriptions, schedules, and prerequisites.

**Note:** This is a cleaned-up public portfolio version of a project I originally built for a university course.

## Problem

Course catalogs usually have the information students need, but not always in a way that feels easy to search. Even when the answer is there, finding it can take a lot of digging through pages and course listings.

I built this project to explore how a conversational interface could make that information easier to access.

## Features

- Search course catalog data through a chat-based interface
- Support questions about:
  - course descriptions
  - schedules
  - prerequisites
  - instructors
  - related course details

## My Role

I built this project as part of a course and used it as a way to explore backend development, language-based interaction, and information access.

My work focused on:
- building the backend in Go
- using an LLM to process and answer user questions
- structuring responses in order to produce a more consistent and useful experience
- thinking about how to make course information easier to navigate from a student’s perspective
